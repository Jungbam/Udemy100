- HTML은 콘텐츠와 페이지 구조를 정의하고 설명하는데 사용

6. HTML 요소에서 CSS를 주는 것은 '인라인 스타일'이라고 함.

7. 스타일 속성을 이용해서 일괄적으로 CSS를 부여할 떄에는
    p{
        font-family : sans-serif;
        text-align : center;
    }
    와 같은 양식이 됨.
    - 각 CSS 속성 + 값은 세미콜론으로 끝나야 함.(한줄에 하나의 속성씩 입력)

14. 구글 폰트에서 폰트 따오기[day04]
    - 폰트 설정 : 구글 폰트에서 따오는데 이때에는 필요한 폰트의 설정을 한 후 그 값에 대한 것만 따와야함.
    - 그러지 않을경우 속도가 늦어지기 때문.
    - font-family: 'Nanum Brush Script', sans-serif;
        , 뒤의 값은 어떠한 이유로 지정한 폰트(위의 Nanum Brush Script)를 가져오지 못했을 때에 default값으로 주어주는 것.

15. 의사 선택자 : 특정 조건을 충족하는 요소의 스타일을 설정할 수 있는 : hover 와 같은 css 선택자

16. HTML 구성
    - body : 콘테이너
    - 부모-자식 관계 : 부모의 스타일을 자식이 따름.
                        cf) a 속성 : 브라우저에서 기본으로 지정한 속성이 있어서임.
                    하지만 부모와 자식 스타일이 다를 경우 자식만의 개성을 가짐.
        * 들여쓰기에 따라 종속되어 있음.

    가. 상속 : 특정 컨테이너의 규칙이 하위 속성에 적용됨.

    나. 계단식 상속 로직 : 하나의 요소에 여러 규칙을 적용한다.
                부모가 특성을 주지만 자식이 지정하는 개성이 우선이 됨.
                같은 단계의 경우 최근에 생긴 속성값을 따름.

    다. 우선순위 : 더욱 구체적인 선택자에게 우선된다.
                결국 자식이 우선적이다.
